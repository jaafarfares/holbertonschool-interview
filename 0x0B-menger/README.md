# 0x0B. Menger sponge






#### This project involves implementing a program that draws a 2D Menger Sponge. The Menger Sponge is a fractal structure that can be created by dividing a cube into smaller cubes and removing the center cube from each division. In this case, you are tasked with drawing a 2D representation of the Menger Sponge.

Here are the key requirements and details of the project:

#### General Requirements:
* Editors allowed: vi, vim, emacs
* Compilation environment: Ubuntu 14.04 LTS
* Compilation flags: gcc 4.8.4 with -Wall, -Werror, -Wextra, and -pedantic
* Coding style: Follow the Betty style, checked using betty-style.pl and betty-doc.pl
* Restrictions: No global variables, no more than 5 functions per file
* Header file: Prototypes of all functions should be included in a header file named menger.h
* Header files should be include guarded
Task:
0. 2D Menger sponge (50 points)
Write a function void menger(int level) that draws a 2D Menger Sponge.
The level parameter specifies the depth of the Menger Sponge. If the level is lower than 0, the function should do nothing.
You are allowed to use the math library, and the program will be compiled using the ld flag -lm.
The size of a level N Menger sponge is calculated as 3^N.
Examples:
A level 0 sponge is a simple 1x1 square (# character).
A level 1 sponge is a 3x3 square of level 0 sponges, except for the center one, which is left empty.
A level 2 sponge is a 3x3 square of level 1 sponges, except for the center one, which is left empty.

Jaafar@~/0x0B-menger$ cat 0-main.c 
#include <stdio.h>
#include <stdlib.h>

#include "menger.h"

/**
 * main - Entry point
 *
 * @ac: Arguments counter
 * @av: Arguments vector
 *
 * Return: EXIT_SUCCESS or EXIT_FAILURE
 */
int main(int ac, char **av)
{
    int level;

    if (ac < 2)
    {
        fprintf(stderr, "Usage: %s level\n", av[0]);
        return (EXIT_FAILURE);
    }

    level = atoi(av[1]);
    menger(level);

    return (EXIT_SUCCESS);
}
